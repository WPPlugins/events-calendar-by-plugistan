== Installation ==

This plugin is very easy to install:

simply:

1. Upload `events-calendar-by-plugistan` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There should be new Menu Item visible - Events Calendar and Scheduler.
